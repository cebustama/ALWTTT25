# planning/active/ — ALWTTT

This folder contains the roadmaps and future-work documents that still matter now.

What belongs here:
- the current combat/gameplay roadmap
- active migration plans if the migration is still in progress
- short/medium-horizon sequencing docs

What does **not** belong here:
- current implemented truth
- archive-only roadmaps
- raw research or freeform brainstorming

## Conflict rule

If this folder conflicts with any active SSoT, the SSoT wins.
If it conflicts with `CURRENT_STATE.md`, treat `CURRENT_STATE.md` as the operational snapshot and then repair the roadmap later.
