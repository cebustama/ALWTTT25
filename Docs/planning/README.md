# planning/ — ALWTTT

This folder contains planning-only material.

It is split into:
- `planning/active/` for roadmaps that still matter now
- `planning/archive/` for completed or superseded planning
- any compatibility subpaths retained temporarily during normalization

What does **not** belong here:
- current subsystem truth
- current runtime contracts
- current authoritative gameplay rules

## Conflict rule

If a planning doc conflicts with a subsystem SSoT or `CURRENT_STATE.md`, the planning doc loses.

## Update rule

Update this folder when:
- deciding next steps
- drafting future architecture
- preserving useful ideas that are not yet governed truth
- archiving a superseded roadmap

## Normalization rule

The primary home for live planning is now `planning/active/`.
Older planning paths should only remain as redirects or migration compatibility shims.
