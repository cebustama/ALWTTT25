# archive/absorbed/ — ALWTTT

This subfolder tracks documents whose useful truth has already been absorbed into governed SSoTs or other governed homes.

Use this area for:
- supersession maps
- redirect notes
- migration crosswalks
- source-to-target traceability

Do **not** use it for:
- live subsystem specs
- active roadmaps
- new design notes

## Conflict rule

Everything in `archive/absorbed/` is historical / traceability-only.
If it disagrees with an active governed doc, the governed doc wins.

## Update rule

Update this folder when a migration batch:
- absorbs a pre-governance source doc into a governed target
- needs a durable redirect record
- closes an authority collision
