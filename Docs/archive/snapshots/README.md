# archive/snapshots/ — ALWTTT

This folder is the documentary home for snapshot policy and any future in-tree snapshot metadata.

The large pre-governance snapshot itself does **not** need to be recopied into the governed tree by default.
Keeping the full snapshot externally is acceptable as long as:
- the governed docs are now the active authority,
- the supersession map is current,
- and the snapshot can still be recovered if historical comparison is needed.
